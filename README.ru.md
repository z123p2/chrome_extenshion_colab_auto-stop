# Colab Auto-Disconnect

[English](README.md) | 🌐 **Русский**

Расширение Chrome, которое автоматически отключает среду выполнения Google Colab после завершения всех ячеек.

## Как это работает

1. `MutationObserver` отслеживает изменения в shadow root `colab-execution-status`
2. Когда ячейки завершаются (пропадает спиннер), скрипт нажимает **Среда выполнения** -> **Отключиться от среды выполнения и удалить ее**
3. Когда появляется диалог подтверждения, скрипт нажимает **Да**
4. Среда выполнения отключена - никаких ручных действий

Расширение реагирует только на `childList` и `attributeFilter: ['open']` - никакого поллинга, никакой лишней нагрузки на страницу.

## Установка

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное**
4. Выберите папку `chrome_extenshion_colab_auto-stop`

## Почему такой подход

Расширение использует только:
- `MutationObserver` на shadow root (без `setInterval`)
- Один `setTimeout(300ms)` для инициализации Closure-меню
- Флаг `pendingDisconnect` чтобы не мешать ручному управлению

Нет `background.js`, нет разрешения `downloads`, нет инлайн-скриптов.
