# Colab Auto-Disconnect

[English](README.md) | 🌐 **Русский**

Расширение Chrome, которое автоматически отключает среду выполнения Google Colab сразу после завершения всех ячеек, предотвращая фоновое списание лимитов.

Работает в **активных и фоновых вкладках** - не нужно держать вкладку Colab в фокусе.

## Зачем это нужно

После завершения кода Google Colab не отключает виртуальную машину - она переходит в режим ожидания (Idle). В это время платные вычислительные единицы (Compute Units) или бесплатные суточные лимиты продолжают расходоваться. Если оставить вкладку на ночь, можно полностью обнулить баланс. Расширение решает эту проблему на уровне браузера один раз для всех блокнотов.

## Как это работает

Два параллельных механизма обеспечивают отключение в любом состоянии вкладки:

**1. MutationObserver (активная вкладка)**
Следит за изменениями в shadow root элемента `colab-execution-status`. Когда ячейки завершаются (пропадает спиннер), скрипт нажимает **Среда выполнения** -> **Отключиться от среды выполнения и удалить ее**, затем подтверждает **Да** в диалоге.

**2. Web Worker (фоновая вкладка)**
Chrome замедляет DOM-таймеры в неактивных вкладках. Web Worker работает в отдельном потоке и срабатывает каждые 10 секунд - Chrome никогда его не throttлит. Worker вызывает `doDisconnect(true)`, которая пропускает проверку `isConnected` (shadow root заморожен в фоновых вкладках) и сразу выполняет отключение.

После успешного отключения флаг `workerDisconnected` блокирует повторные попытки до нового запуска ячеек.

**3. Dialog watcher**
Постоянный `MutationObserver` с `attributeFilter: ['open']` ловит диалог подтверждения. Он нажимает "Да" только когда `pendingDisconnect = true` (вызвано скриптом), игнорируя диалоги, открытые вручную.

## Важные ограничения

- **Удаление данных:** Сессия уничтожается полностью. Все несохраненные файлы в папке `/content/` сотрутся. Сохраняйте результаты на Google Диск до запуска.
- **Только "Выполнить все":** Скрипт реагирует на отсутствие активных ячеек. Если запустить одну ячейку вручную для теста, расширение посчитает это финишем и закроет сессию. Запускайте код целиком через `Ctrl + F9`.
- **Язык интерфейса:** Поиск кнопок привязан к тексту меню. Поддерживаются **Русский** и **Английский** языки интерфейса Colab.

## Установка

1. Скачайте репозиторий и распакуйте архив.
2. Перейдите в браузере по адресу `chrome://extensions/`.
3. Включите **Режим разработчика** (вверху справа).
4. Нажмите **Загрузить распакованное** (вверху слева) и выберите папку `chrome_extenshion_colab_auto-stop`.

## Технические детали

- **Нет `background.js`** - нет service worker, нет лишних разрешений
- **Нет поллинга** - MutationObserver для активных вкладок, Web Worker для фоновых
- **Флаг `pendingDisconnect`** - предотвращает конфликты при ручном управлении сессией
- **Флаг `workerDisconnected`** - предотвращает повторное отключение уже отключенной сессии
- **15-секундный таймаут** - если клик по меню не вызвал диалог, скрипт повторяет попытку
- **Google Closure события** - пункты меню кликаются через `mousedown` + `mouseup` + `click` с реальными координатами (Closure игнорирует обычный `.click()`)
